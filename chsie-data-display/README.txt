This is a revision of CHSIE Data Tables, with data display and export capabilities.


== Description ==

This plugin provides CHSIE administrators with data display and export capabilities focused on user demographic data, the LearnDash LMS, and GDPR compliance.  It also manages data collection popups, when appropriate.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the 'chsie-data-display' folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
